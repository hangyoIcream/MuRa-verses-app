# Verses Android App (Kotlin + Jetpack Compose)

This is a minimal Android app that fetches per-verse JSON from a raw HTTP URL (e.g. GitHub raw).

Important: edit MainActivity.kt and change `baseUrl` to your GitHub raw URL:
https://raw.githubusercontent.com/YOURUSER/YOURREPO/main/data/json/

Build: push to GitHub and use the included GitHub Actions workflow to produce an APK.
